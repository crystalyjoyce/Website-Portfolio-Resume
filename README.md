# Website-Portfolio-Resume
A Website Portfolio Resume is a personal online platform that showcases your skills, experiences, projects, and achievements in a professional and visually engaging way. It serves as both a resume and a portfolio, allowing potential employers or clients to easily view your work, learn about your background, and contact you for opportunities.

---

How to Run Locally
1. Download or clone this repository.
2. Open the project folder in Visual Studio Code.
3. Install the Live Server
4. Right-click the 'indez.html' file -> click "Open with Live Server".
5. The website will open in your default browser.
